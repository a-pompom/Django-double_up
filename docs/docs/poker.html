Poker

Pokerアプリ

ポーカーで遊べるアプリ。役を作成できた場合、ダブルアップを続けてプレイ可能とする。

画面定義

ポーカー画面

処理定義

処理概要

  • STARTボタンでゲーム開始
  • カードを5枚ランダムで表示
  • ホールドする対象を選択し、OKがクリックされると、ホールドされていなかったものを別のカードに置き換え、役が成り立つか判定
  • 役が成り立たない場合は、その時点でゲーム終了
  • 役が成り立った場合、役に応じたコインを獲得し、獲得したコインをベースにダブルアップに挑戦可能となる
  • ダブルアップに挑戦しない場合は、役に応じたコインがそのまま報酬として得られる
  • ダブルアップに挑戦する場合、ダブルアップで敗北した場合、全てのコインは没収され、勝利して終了した場合、勝利回数に応じてコインを獲得
  • 1.初期表示

初期表示としては、伏せられたカードを一枚のみ表示。 STARTボタンがクリックされると、ゲームが開始され、ランダムに5枚のカードを表示。

  • 2.カード選択

ユーザは、5枚のカードの内、残しておきたいカードを「ホールド対象」として選択。ホールド対象を選択し終わり、 「OK」をクリックすると、選択完了とする。

  • 3.役判定

ホールドされたカードはそのまま、ホールドされなかったカードは、再度ランダムに生成し、結果セットとしての5枚のカードを決定。 5枚のカードから、役が成立するかどうかを判定。 役の種類は以下。

  • ツーペア
  • スリーカード
  • ストレート
  • フラッシュ
  • フルハウス
  • フォーカード
  • ストレートフラッシュ
  • 4.結果表示

役判定の結果を画面へ表示。役が成立した場合は、役ごとのレートに応じたコインを獲得。役が成立しない場合は、コインを没収。 そして、役が成立した場合のみ、ユーザは、ポーカーで獲得したコインを種に、ダブルアップに挑戦が可能となる。

ダブルアップは、任意の回数挑戦可能となり、一度勝利する度に、獲得したコインの倍のコインが報酬として得られるようになる。 終了のタイミングは、ユーザが決定し、終了する前にダブルアップで敗北した場合、コインは全て没収となる。

ダブルアップ終了後は、ポーカーから再び挑戦可能となる。
